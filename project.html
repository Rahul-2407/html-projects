Empowering young aspects to build their own gaming software and to experience variety of AAA games

The Epic Games Launcher is a desktop application developed by Epic Games that serves as a digital storefront and game library manager. It provides access to the Epic Games Store, where users can purchase and download a variety of games ranging from indie titles to AAA releases.


A popular feature is the regular offering of free games, which can be permanently claimed during specific time periods. Additionally, the launcher allows for easy game library management, enabling users to download, install, and update their games seamlessly.


The main use of this software is divided into 3 different parts:

  • A single environment for gaming
  • multiplayer support
  • public forum
